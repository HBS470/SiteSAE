Exercice 1

compiler et exécuter un premier programme

A l'aide d'un éditeur de texte (utilisez par exemple gedit ou sublime text qui sont installés sur les machines de l'IUT), copiez le programme Java ci-dessous.

 

/* Bonjour.java : ce programme affiche un message de bienvenue */

class Bonjour {

 

  public static void main(String[] args) {

    System.out.println("Bonjour !");

  }

 

}

 

Compilez et exécutez ce programme :

javac Bonjour.java

java Bonjour

 

Remarques :

  • •Vous pouvez lancer gedit en tapant gedit & dans le terminal de façon à ne pas bloquer la fenêtre xterm, ce qui vous permet d'y compiler et exécuter votre programme.  

  • •Vérifiez que gedit est configuré pour qu'il affiche en couleur la syntaxe (dès que vous aurez donné un nom avec l'extension .java à votre fichier, gedit saura qu'il est écrit en java). 

  • •Configurez gedit pour qu'il n'aille pas à la ligne automatiquement. 

  • •Configurez gedit pour qu'il affiche les numéros de lignes.  

  • •Sauvegardez régulièrement votre programme : cela vous évitera de le perdre en cas de mauvaise manipulation ou de plantage du système. 

     

Regardez les fichiers présents dans votre dossier (avec la commande ls -l). Un fichier Bonjour.class a été créé par le compilateur Java (lorsque vous avez exécuté la commande  javac Bonjour.java). C'est ce fichier que l'on exécute à l'aide de la machine virtuelle Java en tapant la commande :  java Bonjour

Exercice 3

Saisie au clavier – int – instruction conditionnelle

1ère partie : étudier un exemple

Pour réaliser la saisie d’une information au clavier, nous allons utiliser en Java un Scanner. Pour l’instant, ne cherchez pas trop à comprendre ce dont il s’agit... essayez juste d’apprendre à l’utiliser.

Récupérez sur Moodle le programme Bienvenue.java qui contient des exemples de saisies.

Vous remarquerez avant la déclaration de classe, la ligne :

 import java.util.Scanner;

 

Elle est nécessaire pour utiliser un Scanner Java.

Vous remarquerez dans le programme la ligne :

 Scanner scanner = new Scanner(System.in).useDelimiter("\n");

Elle permet de créer un Scanner et de le stocker dans une variable nommée scanner de type Scanner.

Nous pouvons ensuite réaliser la saisie d’un texte (type String) avec scanner.nextLine() et la saisie d’un nombre entier (type int) avec scanner.nextInt().

Remarque :

  • •Le type int permet de manipuler des nombres entiers. 

  • •Lorsqu’on a terminé d’utiliser un Scanner, il est préférable de le fermer (scanner.close() à la fin du programme). 

 

2ème partie : travail à faire

 

Modifiez le programme pour que dans le cas où l’utilisateur saisit pour l’âge un nombre négatif il affiche « Vous n’êtes pas encore né ! ». De préférence, faites en sorte que rien d’autre alors ne s’affiche.

 

Remarque : lorsqu’un if ou un else contient plusieurs instructions, il est nécessaire en Java de mettre des accolades. Par exemple :

 if (age == 18) {

  System.out.println("Vous êtes majeur.");

  System.out.println("Et félicitations vous avez 18 ans !");

 }

 

Remarque : en Java, pour une condition d’égalité, il faut utiliser un double signe égal (comme dans l’exemple ci-dessus).

Exercice 5

Circonférence et aire d'un cercle

Développez un programme dans lequel vous déclarez et initialisez une constante PI égale à 3.14. Ce programme doit demander la saisie au clavier de la valeur du rayon r d'un cercle, puis calculer et afficher la circonférence (2*PI*r) et l'aire (PI*r*r) de ce cercle.

 

Remarque 1 : La valeur d'une constante est... constante (elle ne peut pas être modifiée pendant l'exécution du programme). Pour déclarer une constante, utilisez la syntaxe (à placer en dehors de la méthode main) :

 final static double PI=3.14;

 

Remarque 2 : en Java les noms de constantes sont habituellement écrits entièrement en majuscules (il s'agit d'une règle de bonne pratique de programmation Java).
